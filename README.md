# 基于Orange Pi Kunpeng Pro 的无人机港口集装箱智能检测与计数系统

本项⽬基于Orange Pi Kunpeng Pro，结合 YOLOv8n ⽬标检测模型，构建集装箱实时检测与计数系统，实现对无人机拍摄视频中的集装箱的⾃动识别与数量统计。交互界面由PyQt5构建。
